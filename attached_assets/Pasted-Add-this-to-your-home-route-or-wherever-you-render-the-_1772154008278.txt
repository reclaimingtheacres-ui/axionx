Add this to your home() route (or wherever you render the dashboard) after you open the DB cursor:

# Job counts for dashboard tiles
cur.execute("SELECT COUNT(*) AS c FROM jobs")
jobs_all = cur.fetchone()["c"]

cur.execute("SELECT COUNT(*) AS c FROM jobs WHERE status = 'New'")
jobs_new = cur.fetchone()["c"]

cur.execute("SELECT COUNT(*) AS c FROM jobs WHERE status LIKE 'Active%'")
jobs_active = cur.fetchone()["c"]

cur.execute("SELECT COUNT(*) AS c FROM jobs WHERE status = 'Suspended'")
jobs_suspended = cur.fetchone()["c"]

cur.execute("SELECT COUNT(*) AS c FROM jobs WHERE status = 'Invoiced'")
jobs_invoiced = cur.fetchone()["c"]

Then pass these into your render call:

return render_template(
    "home.html",
    jobs_all=jobs_all,
    jobs_new=jobs_new,
    jobs_active=jobs_active,
    jobs_suspended=jobs_suspended,
    jobs_invoiced=jobs_invoiced
)
