Step 1: Stage CSV Data

Upload the GeoOp jobs.csv and notes.csv exports. Data is loaded into staging tables for review before importing. The Description field is parsed to extract client, account, regulation type, amounts, security details, registration, VIN, and delivery instructions.

Attachment Scan: If the GeoOp attachments/ folder has been uploaded to the server, enter the path below to scan and index files by Job ID and Note ID. Structure: attachments/{job_id}/{note_id}/{file_id}/{filename}

Step 2: Review Diagnostics

Click "Load Diagnostics" to generate a report from the staged data.

Step 3: Execute Import

Import staged data into AxionX. Jobs, customers, security items, and optionally notes will be created. Existing jobs matched by reference number or GeoOp Job ID will be skipped (insert-only mode) or updated.
